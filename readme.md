AeroBurst
“AeroBurst- Run anywhere. Store in R2. Pay nothing to us."
Aero-Burst is an R2-centric multi-cloud compute bursting platform that enables developers to execute arbitrary computational workloads on ephemeral cloud virtual machines. Users bring their own cloud accounts (AWS, Hetzner), pay providers directly, and use Cloudflare R2 for all storage operations to eliminate egress costs.
